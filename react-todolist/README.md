# 📋 Trello Clone (Kanban Board)

`React-Beautiful-DnD`를 활용하여 직관적인 드래그 앤 드롭 기능을 구현한 칸반 보드 형식의 투두리스트입니다.

## ✨ 핵심 기능

- **드래그 앤 드롭 (Drag & Drop)**: `react-beautiful-dnd`를 사용하여 보드 내 순서 변경 및 보드 간 카드 이동을 자유롭게 할 수 있습니다.
- **보드 관리**: 새로운 작업 카테고리(Board)를 추가하고 관리할 수 있습니다.
- **폼 핸들링**: `react-hook-form`을 통해 입력값 검증(Validation) 및 카드 생성을 효율적으로 처리합니다.
- **상태 유지 (Persistence)**: `recoil-persist`를 활용하여 브라우저를 새로고침해도 데이터가 로컬 스토리지에 유지됩니다.

## 🛠 Skills

- **React**: 컴포넌트 기반 설계를 통해 사용자와 실시간으로 상호작용하는 UI를 효율적으로 구현했습니다.
- **TypeScript**: 코드 작성 단계에서 타입 오류를 사전에 방지하고, 자동 완성 기능을 적극 활용하여 개발 생산성과 코드의 안정성을 높였습니다.
- **Styled-components**: CSS-in-JS 방식으로 스타일을 관리하여 재사용 가능한 컴포넌트를 설계하고, Props에 따른 동적 스타일링을 구현했습니다.
- **Recoil**: 전역 상태 관리를 담당하며, `recoil-persist`를 사용하여 `localStorage`에 데이터를 저장함으로써 새로고침 후에도 상태가 유지되도록 설계했습니다.
- **React-beautiful-dnd**: 복잡한 드래그 앤 드롭 로직을 라이브러리를 통해 안정적으로 구현했습니다.

## 💡 주요 학습 내용

- **상태 관리 최적화 (Recoil)**: 중첩된 객체 구조(`{ 카테고리: [데이터] }`)를 설계하고, `recoil-persist`로 브라우저 종료 후에도 데이터가 유지되도록 구현했습니다.

- **DnD 로직 구현**: `onDragEnd` 시점에서 배열의 불변성을 유지하며 데이터를 재정렬하는 알고리즘을 구현하고, 보드 간 이동 로직을 처리했습니다.
- **Form Validation**: `React Hook Form`의 비제어 컴포넌트 방식을 활용해 불필요한 리렌더링을 줄이면서도 실시간 유효성 검사를 수행하는 방법을 배웠습니다.
- **성능 최적화 (React Hook Form)**: 비제어 컴포넌트 방식을 사용해 불필요한 리렌더링을 줄이고, 선언적인 유효성 검사로 코드 가독성을 높였습니다.
